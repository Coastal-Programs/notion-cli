`notion-cli help`
=================

Display help for notion-cli.

* [`notion-cli help`](#notion-cli-help)
* [`notion-cli <command> --help`](#notion-cli-command---help)

## `notion-cli help`

Display help for notion-cli. Run with no arguments to see all available commands.

```
USAGE
  $ notion-cli --help
  $ notion-cli -h

DESCRIPTION
  Display the list of all available commands and global flags.
```

## `notion-cli <command> --help`

Display help for a specific command or subcommand.

```
USAGE
  $ notion-cli <command> --help
  $ notion-cli <command> -h

EXAMPLES
  Show help for db commands

    $ notion-cli db --help

  Show help for db query subcommand

    $ notion-cli db query --help

  Show help for page create

    $ notion-cli page create --help

  Show version information

    $ notion-cli --version
```
